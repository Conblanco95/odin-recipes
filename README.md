# odin-recipes
Website creation for Foundations HTML project exercise.

## Summary of project
This project is intended to showcase the HTML skills obtained so far in the Foundations chapter of T.O.P.

## Skills to showcase
* Boilerplate setup with proper use of the `<head>` element and it's containing `<meta>` information.
* Proper use of HTML elements, including (but not limited to); `<img>`, `<a>`, `<h1> --> <h6>`, `<p>`, etc.
* Proper use of list items (unordered and ordered).
* Use of relative web pages within the project folder and linking.
